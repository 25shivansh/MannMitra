## 🌟 Overview
**Mann Mitra** is a **Digital Mental Health and Psychological Support System** designed for **students in higher education institutions**.  
It bridges the gap between **technology and emotional wellbeing**, offering accessible, inclusive, and affordable mental health support through a blend of **AI-driven assistance** and **human counseling**.

---

## 💡 Core Features

### 🤖 AI Chat & Voice Therapist
- Provides **24/7 emotional support** via chat and voice.  
- Built on advanced language models for empathetic, natural conversations.

### 🧩 Scientific Screening Tools
- Supports **PHQ-9**, **GAD-7**, and **GHQ** assessments.  
- Enables early detection of **depression**, **anxiety**, and other mental health conditions.

### 🌐 Multilingual Accessibility
- Multi-language interface to support students from **rural and semi-urban regions** across India.  
- Makes mental health care more inclusive and accessible.

### 👩‍⚕️ Counselor Booking System
- Allows students to **book appointments** with certified human counselors.  
- Ensures a seamless handoff from AI assistance to professional therapy when required.

### 🗂️ Automated Session Reports
- Generates secure, automated reports after every chat, screening, or voice therapy session.  
- Shares reports with authorized college counselors for **proactive intervention**.

## 🚀 Getting Started

### Prerequisites
- Node.js ≥ 18  
- npm or yarn  
- MongoDB connection (local or cloud)

### Installation
```bash
# Clone the repository
git clone https://github.com/<your-username>/mann-mitra.git

# Navigate to project directory
cd mann-mitra

# Install dependencies
npm install

# Start the development server
npm run dev
